# App Abuelos

Aplicación Flutter extremadamente simple y accesible para adultos mayores.

Pantallas:
- Ayuda: botón grande "Enviar Alerta" con SnackBar.
- Pendientes: lista editable con guardado local (SharedPreferences).
- Calendario: calendario simple con selección de fecha (table_calendar).

Requisitos:
- Flutter SDK instalado.

Primeros pasos:
1) Entra a la carpeta del proyecto:
   `cd app_abuelos`
2) Genera los archivos de plataforma si aún no existen:
   `flutter create .`
3) Instala dependencias:
   `flutter pub get`

Ejecutar:
- Web: `flutter run -d chrome`
- Android: `flutter run -d android`
- iOS: `flutter run -d ios`
